💡 HOW IT WORKS
═══════════════════════════════════════════════════════════════════════════

1. USER ENTERS QUERY
   "Patient with chest pain and shortness of breath"
                    ↓
2. SYSTEM CONVERTS TO EMBEDDING
   Text → Vector [0.23, -0.45, 0.12, ..., 0.88]
                    ↓
3. SIMILARITY SEARCH
   Find most similar cases in vector DB
                    ↓
4. RETRIEVE TOP CASES
   Get 5 most similar cases with scores
                    ↓
5. GENERATE RESPONSE
   Create HTML summary with findings
                    ↓
6. DISPLAY RESULTS
   User sees similar cases + analysis
